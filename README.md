# erldb

ORM (Object-relational mapping) application implemented in Erlang.

Right now we are aiming todo a 0.1 release. When it is done we will

# Build
To build erldb use Makefile to build.

```
  $ make
```

## Make documentation
To generate documentation of erldb:

```
  $ make docs
```
